# Ficha RPG (skeleton)

Pequeno esqueleto Svelte que implementa login simples e render read-only de ficha definida por JSON.

Quick start:

```bash
npm install
npm run dev
```

Arquivos principais:
- `index.html` — ponto de entrada
- `src/App.svelte` — lógica de login e render
- `users.json`, `sheets/sheet1.json` — exemplos de dados
